Nth() Page Selector

The 1st page header/footer should be:

:first* :nth(1) :nth(odd)
:nth(-n+3) Page 1*

* If this is not the first doc, “:first” will not appear, and the Page numbering will be consecutive from the previous doc. A blank page may be inserted by :root { break-before: recto; }.

The 2nd page header/footer should be:

:nth(3n+2) :nth(even)
:nth(-n+3) :nth(3n-1) Page 2*

The 3rd page header/footer should be:

:nth(3n) :nth(odd)
:nth(-n+3) Page 3*

The 4th page header/footer should be:

:nth(even)
Page 4*

The 5th page header/footer should be:

:nth(3n+2) :nth(odd)
:nth(n+5) :nth(3n-1) Page 5*

The 6th page header/footer should be:

:nth(3n) :nth(even)
:nth(n+5) Page 6*

The 7th page header/footer should be:

:nth(odd)
:nth(n+5) Page 7*

The 8th page header/footer should be:

:nth(3n+2) :nth(even)
:nth(n+5) :nth(3n-1) Page 8*

The 9th page header/footer should be:

:nth(3n) :nth(odd)
:nth(n+5) Page 9*
